# my-utils
## 说明
自用的函数工具库,集合了他人和自己造的轮子, 用于前端和node后端

尽量支持多版本、兼容适配各种场景（如编写js、ts版本，多写几个重载函数）

可按需复制粘贴，也可复制整个utils文件夹

## 规范
- lint: 自定义 **eslint**
- 测试用例： 按需测试，涉及风险高的才写。目前的方案是手动测试，后期预计引入单元测试框架。
- 模块化: 统一使用 **commonJs** 语法
- 书写规范: 
```js
/**
 * 功能简介
 * /

// 函数说明
/* 代码 */
```
- 提交规范： commit里写上功能
